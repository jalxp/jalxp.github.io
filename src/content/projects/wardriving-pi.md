A tiny Linux box that listens more than it talks.

## What it does

- Logs Wi-Fi SSIDs, BSSIDs, and signal strength as I move around.
- Serves as a portable playground for Kismet, tcpdump, and friends.
- Runs whatever Python scripts I need for whatever rabbit hole I'm currently in.

## Why

Because the best way to understand networking is to watch real traffic, and the
best way to understand Wi-Fi specifically is to see it from the air. And because
carrying a laptop around for this is overkill.

## Status

Hardware is set up; software is a living pile of scripts. Getting cleaner over
time.
