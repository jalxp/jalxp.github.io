GoodBarber is a no-code app builder. Every customer app — and there are thousands
live on the App Store — is built on top of a shared iOS engine. That engine is
where I've spent the last 5+ years.

## What I work on

- **User-facing features** — token-based authentication, push notifications, caching strategies, offline behavior.
- **Design system** — expanded the component library and migrated screens to a consistent visual language.
- **Third-party integrations** — Stripe, Google Ads, Meta Events, Lottie.
- **CI/CD** — Bitrise workflows, TestFlight pipelines, reducing deployment friction.
- **AI tooling** — local LLMs, MCP servers, custom agents for code generation and docs.
- **Mentoring** — onboarding guides, bringing up interns and juniors, representing the company at university career fairs.

The code isn't public, but the apps that ship on top of it are.
